# AptiVerbal Pro

AptiVerbal Pro is a placement preparation web app for aptitude, verbal, reasoning, coding aptitude, company model papers, and interview practice.

## Project Structure

- `index.html` - Main frontend page and app layout.
- `css/styles.css` - All frontend styling.
- `js/app.js` - Quiz navigation, scoring, completion tracking, and UI behavior.
- `data/question-data.js` - Topic data, company model-paper data, and question banks.
- `server/server.js` - Node.js static file server.
- `server.js` - Easy root launcher for the server.

## Features

- Topic-wise practice questions.
- Previous question, next question, direct question number navigation, and final submission.
- Correct, incorrect, skipped, and detailed answer review.
- Saved completed status in the browser.
- Company-wise model papers for MNC placement preparation.
- Round 1 written test, Round 2 technical MCQ, and Round 3 coding/interview practice.
